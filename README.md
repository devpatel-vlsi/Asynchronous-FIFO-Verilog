# Asynchronous FIFO in Verilog

## Overview
Designed an asynchronous FIFO using dual-port RAM with independent read and write clock domains.

## Features
- Gray-code pointer synchronization
- Full and Empty flag generation
- Clock Domain Crossing (CDC)

## Tools Used
- Verilog
- Vivado
- EDA Playground

## Results
Verified functionality through simulation and waveform analysis.
